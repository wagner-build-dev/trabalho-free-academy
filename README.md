## Como Instalar e Executar
1. Instale o XAMPP
Baixe em: https://www.apachefriends.org/

2. Baixe e Copie os arquivos
Baixe todos os arquivos no link: https://github.com/wagner-build-dev/trabalho-free-academy,
crie uma pasta chamada "free-academy" dentro de "C:\xampp\htdocs\" e renomeie para "free-academy".

### 3. Crie o banco de dados
- Abra o XAMPP e inicie **Apache** e **MySQL**
- Acesse: http://localhost/phpmyadmin
- Clique em **Importar** → selecione o arquivo que esta nesse caminho:"C:\xampp\htdocs\free-academy\bd\comercio_eletronico.sql"
   → Execute


### 4. Acesse o site
http://localhost/free-academy/
